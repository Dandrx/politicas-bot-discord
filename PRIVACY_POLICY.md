# Politica de Privacidad

Ultima actualizacion: 19 de junio de 2026

Esta politica explica como se manejan los datos del bot de Discord para ventas de optimizacion de PC.

## Datos que recopila el bot

El bot puede guardar la siguiente informacion cuando una persona usa el panel, solicita una recomendacion o abre un ticket:

- ID de usuario de Discord.
- Nombre de usuario de Discord.
- Componentes del PC enviados por el usuario, como CPU, GPU, RAM y almacenamiento.
- Uso principal, problemas reportados, presupuesto o notas enviadas por el usuario.
- Plan de optimizacion recomendado.
- ID del canal de ticket y estado del ticket.
- Fecha y hora de creacion del lead o ticket.

## Para que se usan los datos

Los datos se usan solamente para:

- Recomendar una optimizacion segun los componentes del usuario.
- Crear y administrar tickets de soporte o venta.
- Dar seguimiento a clientes interesados.
- Mejorar el proceso de atencion dentro del servidor de Discord.

## Donde se guardan los datos

Los datos se guardan en una base de datos local SQLite del bot. No se venden ni se comparten con terceros.

## Quien puede ver los datos

Los datos pueden ser vistos por el propietario del bot y por el equipo autorizado del servidor, especialmente el personal encargado de ventas o soporte.

## Eliminacion de datos

Un usuario puede solicitar la eliminacion de sus datos contactando al propietario del servidor o al equipo de soporte dentro del Discord donde se usa el bot.

## Seguridad

Se aplican permisos de Discord para limitar el acceso a tickets y canales internos. Aun asi, ningun sistema es completamente infalible, por lo que los usuarios no deben enviar contrasenas, tokens, datos bancarios ni informacion extremadamente sensible por Discord.

## Cambios en esta politica

Esta politica puede actualizarse cuando cambien las funciones del bot o la forma en que se manejan los datos. La fecha de actualizacion indicara la version mas reciente.

## Contacto

Para preguntas sobre esta politica o solicitudes de eliminacion de datos, contacta al propietario del servidor de Discord donde se utiliza este bot.
