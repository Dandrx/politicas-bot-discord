# Condiciones de Servicio

Ultima actualizacion: 19 de junio de 2026

Estas condiciones explican las reglas de uso del bot de Discord para ventas de optimizacion de PC.

Al usar este bot dentro de un servidor de Discord, aceptas estas condiciones.

## Uso del bot

El bot permite:

- Solicitar recomendaciones de optimizacion segun los componentes del PC.
- Ver planes o precios de servicios de optimizacion.
- Abrir tickets de atencion, soporte o venta.
- Enviar informacion tecnica sobre el equipo, como CPU, GPU, RAM, almacenamiento, juegos, FPS o problemas de rendimiento.

## Informacion proporcionada por el usuario

El usuario es responsable de que la informacion enviada sea correcta. Las recomendaciones del bot se basan en los datos proporcionados y pueden no ser exactas si los datos estan incompletos o son incorrectos.

No debes enviar contrasenas, tokens, codigos de seguridad, informacion bancaria ni datos extremadamente sensibles por Discord.

## Recomendaciones y resultados

Las recomendaciones del bot son orientativas. El rendimiento final de una optimizacion puede variar segun el hardware, el estado del sistema, los programas instalados, los drivers, la temperatura, el internet, la configuracion de Windows y otros factores.

El bot no garantiza una cantidad especifica de FPS, una reduccion exacta de latencia ni una mejora fija de rendimiento.

## Servicios de optimizacion

Los servicios, precios, tiempos de entrega y metodos de pago pueden variar segun el servidor o el equipo que opere el bot. Cualquier compra, pago o acuerdo se gestiona directamente entre el usuario y el responsable del servidor.

El bot por si mismo no procesa pagos ni almacena datos bancarios.

## Tickets y soporte

Al abrir un ticket, aceptas que el personal autorizado del servidor pueda ver la informacion enviada dentro de ese canal para atender tu solicitud.

El equipo del servidor puede cerrar tickets inactivos, duplicados, abusivos o que incumplan las reglas del servidor.

## Conducta prohibida

No esta permitido:

- Usar el bot para spam o abuso.
- Enviar informacion falsa con intencion de perjudicar el servicio.
- Intentar explotar, romper o manipular el funcionamiento del bot.
- Usar tickets para acoso, amenazas, fraude o contenido ilegal.
- Compartir datos privados de otras personas sin permiso.

## Suspension o restriccion de uso

El propietario del servidor o el equipo autorizado puede limitar, suspender o bloquear el acceso al bot si un usuario incumple estas condiciones o las reglas del servidor.

## Privacidad

El uso del bot tambien esta sujeto a la Politica de Privacidad publicada para este servicio.

## Cambios en estas condiciones

Estas condiciones pueden actualizarse cuando cambien las funciones del bot, los servicios ofrecidos o las reglas del servidor. La fecha de actualizacion indicara la version mas reciente.

## Contacto

Para preguntas sobre estas condiciones, contacta al propietario o al equipo de soporte del servidor de Discord donde se utiliza este bot.
